{%- comment -%} Group prompts by year at build time using Liquid. IMPORTANT: `map: "year"` is invalid on Ruby Time objects in Liquid 4 — calling time["year"] returns nil, so the timeline would render empty. Instead we do a single O(n) pass to build a pipe-delimited string of unique years, then split it. The inner loop (O(Y×P)) is unavoidable with Jekyll 3.9.x (no group_by_exp). {%- endcomment -%} {%- assign sorted_prompts = site.prompts | sort: "date" | reverse -%} {%- assign year_pipe = "" -%} {%- for prompt in sorted_prompts -%} {%- assign _y = prompt.date | date: "%Y" -%} {%- unless year_pipe contains _y -%} {%- assign year_pipe = year_pipe | append: _y | append: "|" -%} {%- endunless -%} {%- endfor -%} {%- assign years = year_pipe | split: "|" -%}
{%- for year_str in years -%}
{{ year_str }}

{%- endfor -%}
