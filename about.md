**Promptly** is a curated record of creative work made with AI — a place to document prompts, reflect on results, and build a public archive of the human-AI creative process.

## Why keep a prompt portfolio?

The craft of prompting is real. A well-structured prompt is equal parts instruction, intuition, and creative vision. Documenting them is like keeping a sketchbook: patterns emerge, skills compound, and your best thinking is preserved.

## How entries are structured

Each entry captures:

- **The raw prompt** — exactly as it was submitted
- **The model** — which AI system produced the output
- **The result** — image, video, text, or code
- **Context** — notes on intent, iteration, and what worked

## Technical stack

This site is built with [Jekyll](https://jekyllrb.com/) and hosted for free on [GitHub Pages](https://pages.github.com/). No CDN dependencies, no JavaScript frameworks — just a static site that loads instantly and requires zero ongoing maintenance.

The visual system uses pure Sass with CSS custom properties for three runtime themes, switching without a page reload via a radial ink-spread animation.
